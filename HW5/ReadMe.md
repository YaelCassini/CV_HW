# ReadMe

1. 本实验使用 python3.6 ，在tensorflow框架下完成。

2. 源代码为 code目录下的 "HW5_1.py"、"HW5_2.py"。

4. “HW5_1.py”为手写字体识别的训练和测试程序。

5. “HW5_2.py”为物体分类的训练和测试程序。

6. result文件夹中是一些中间结果，由于运行过程中会有图像覆盖写出，因此有些结果未在该文件夹中展示，而只放在报告中。

7. 实验中用到的python库及其版本如下：

   numpy 	1.13.3
   opencv 	3.3.1
   pillow 	8.0.1
   matplotlib 	3.1.1
   
   tensorflow	1.2.1
